object map_aic

Contains a series of methods used for extracting and handling weights to codon values.

Where a codon is an Int between 0-63 representing all the possible combinations of 3 MRNABases.

def extractCodonWeights(input: String): Option[Map[Int, Double]]

This will take a string from a file containing codons (groups of 3 MRNABases) and the weight of each one on each line and deserializes them, including using MRNABases.indexCodon to deserialize the codon.

This will take a string from a file containing codons (groups of 3 MRNABases) and the weight of each one on each line and deserializes them, including using MRNABases.indexCodon to deserialize the codon.
