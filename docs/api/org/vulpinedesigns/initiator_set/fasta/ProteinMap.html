class ProteinMap(val Codons: List[Int], val ProteinBases: List[AminoAcid], val AltAminoAcid: AltProteinBases)

A class for associating a list of mRNA codons to a list of amino acids (a protein).

Value Params
AltAminoAcid

Potential alternative set of amino acids if the codon is at the start or end.

Codons

The list of codons that convert to the given amino acids.

ProteinBases

The list of AminoAcids that match to the codons.
