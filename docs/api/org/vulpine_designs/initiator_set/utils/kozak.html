class KzConsensus(val CodonStart: Int, val Sequence: List[KzNucleotide])

This stores the list or sequence of KzNucleotide, along with a few helper methods and data, allowing for a list of mRNA.Bases to be compared against it to determine if the desired sequence has been found.

This stores the list or sequence of KzNucleotide, along with a few helper methods and data, allowing for a list of mRNA.Bases to be compared against it to determine if the desired sequence has been found.

==Example== You can compare a list of mRNA.Bases against the consensus by using KzConsensus.similarity this returns a double denoting the "strength" of whether or not the sequence is the desired one;

   val similarityScore = consensus.similarity(listOfBases)
