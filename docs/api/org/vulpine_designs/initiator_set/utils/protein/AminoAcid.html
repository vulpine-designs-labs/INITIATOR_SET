AminoAcid

enum AminoAcid

An enum representing all the types of amino acids which in a list make up a protein. This includes AminoAcid.Gap for any gaps in the protein chain, and AminoAcid.Stop + AminoAcid.Start indicating stops and starts of a protein chain.
